# Global Claude Code Instructions

## Commit Guidelines

- Do not include Co-Authored-By lines in commit messages

## Documentation Style

- Never use Unicode box-drawing characters or symbols in markdown or any other files.
  Use ASCII equivalents instead:
  - `+--` instead of `├──` / `└──`
  - `|` instead of `│`
  - `->` instead of `→`
  - `[x]` instead of `✓` / `✅`
  - `[ ]` instead of `✗` / `❌`
  - `[!]` instead of `⚠`
  - `-` instead of `─`
