---
name: ASCII-only text files
description: All text files must use ASCII characters only, never Unicode symbols
type: feedback
---

Always use ASCII characters in all text files (markdown, templates, docs, configs).
Never use Unicode symbols. Apply this to every project without being asked.

Replacements:
- `--` instead of em dash (--)
- `-` instead of en dash
- `->` instead of arrow (->)
- `+--` instead of box-drawing chars
- `|` instead of vertical bar box char
- `[x]` instead of checkmarks
- `[ ]` instead of X/cross marks
- `[!]` instead of warning symbols
- `'` instead of curly/smart quotes
- `"` instead of curly double quotes
- `...` instead of ellipsis
